### Ejecuciones maven:
- Ejecutar con maven: `mvn compile exec:java`
- `mvn clean install`
- `mvn build`
- Verificar sonar: `mvn verify sonar:sonar`
- Generar informes (surefire): `mvn site:site` (se generan en target/site/index.html)
